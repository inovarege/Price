# Ambiens - Painel de Formação de Preços

Plataforma estática para cálculo de formação de preços da **Ambiens Consultoria Ambiental**.

## 🚀 Funcionalidades
- Cálculo de Despesas Diretas (Serviços, Logística, Horas Internas, etc).
- Definição de Taxas Variáveis (Impostos, Comissão, Lucro).
- Comparativo dinâmico entre Preço Sugerido e Preço Praticado.
- Exportação de relatório oficial em PDF.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Design Premium)
- JavaScript Vanilla
- Bibliotecas: Lucide Icons, html2canvas, jsPDF

## 📂 Estrutura do Projeto
- `index.html`: Estrutura da página.
- `styles.css`: Estilização e design system.
- `app.js`: Lógica de cálculos e interatividade.
